# 🌟 My Personal Portfolio

A **full-stack personal portfolio** built to showcase my projects, skills, and experience dynamically.  
The portfolio is fully editable, allowing me to **add, update, or delete content** without touching the code similar to a Facebook style personal page.

---

## 🚀 Features

### Backend (ASP.NET Core)
- Structured with **Clean Architecture**, **DDD**, **CQRS**, and **MediatR**  
- Secure user authentication and authorization (JWT)  
- CRUD operations for projects, experience, skills, education, and testimonials  
- Real-time updates using **SignalR**  
- Fully containerized with **Docker** for easy deployment  
- Unit tests 
- Swagger API documentation for easy testing  

### Frontend (React)
- Built with **React + TypeScript + Tailwind CSS** for a modern, type-safe, responsive UI  
- **Dashboard page**: Overview of projects, tasks, and notifications  
- **Projects page**: Display and manage projects  
- **Project members page**: View and manage team members  
- **Tasks & Assign Task page**: Create and assign tasks dynamically  
- **Calendar page**: Visualize tasks and events (almost complete)  
- Connects to backend via REST API and SignalR for real-time updates
